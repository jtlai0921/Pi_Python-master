請先將picamera置入樹梅派
請參考課本 19-4頁

Enable Camera
請參考課本 19-5(下面)~19-7頁


1.更新套件
sudo apt-get update

2.拍照
raspistill -o image.png

3.安裝各種圖片格式函示庫
sudo apt-get install -y libjpeg-dev libtiff5-dev libjasper-dev libpng12-dev

4.安裝各種影片格式函示庫
sudo apt-get install -y libavcodec-dev libavformat-dev libswscale-dev libv4l-dev
sudo apt-get install -y libavcodec-dev libx264-dev
sudo apt-get install python-picamera


5.安裝開發工具 Cmake
sudo apt-get install build-essential cmake pkg-config

6.安裝OpenCV顯示所需要的GTK程式庫
sudo apt-get install libgtk2.0-dev

7.安裝OpenCV顯示所需要的矩陣程式庫
sudo apt-get install libatlas-base-dev gfortran

8.安裝編譯OpenCV所需要的Python2, 3標頭檔
sudo apt-get install python2.7-dev python3-dev python3.5-dev

9.安裝PythonPackageManager
wget https://bootstrap.pypa.io/get-pip.py --no-check-certificate
sudo python get-pip.py

10.安裝NumPy Open 數值運算程式庫
sudo pip install numpy
sudo pip install --upgrade numpy

11.下載並解壓縮OpenCV與OpenCV_contrib原始檔
cd ~
wget -O opencv.zip https://github.com/Itseez/opencv/archive/3.3.0.zip
unzip opencv.zip
wget -O opencv_contrib.zip https://github.com/Itseez/opencv_contrib/archive/3.3.0.zip
unzip opencv_contrib.zip

12.編譯OpenCV (預估 3~10小時)
cd ~/opencv-3.3.0/
mkdir build
cd build
cmake \
-D CMAKE_BUILD_TYPE=RELEASE \
-D CMAKE_INSTALL_PREFIX=/usr/local \
-D INSTALL_PYTHON_EXAMPLES=ON \
-D OPENCV_EXTRA_MODULES_PATH=~/opencv_contrib-3.3.0/modules \
-D BUILD_EXAMPLE=ON ..
make clean
make

13.安裝 OpenCV
sudo make install
sudo ldconfig

14.測試OpenCV
cd ~
pi@raspberrypi:~ $ python
Python 2.7.9 (default, Sep 17 2016, 20:26:04)
[GCC 4.9.2] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>> import cv2
>>> cv2.__version__
'3.3.0'
>>> exit()

15.清除不需要的檔案
rm -rf opencv-3.3.0 opencv_contrib-3.3.0
